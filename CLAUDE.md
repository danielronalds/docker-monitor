# Docker Benchmark

Single-file Python script that live-streams Docker container stats (memory and CPU) to a matplotlib chart.

## Constraints

- Must remain a single file, runnable via `uv run docker-monitor.py` or `uv run <url>`
- Dependencies declared via PEP 723 inline script metadata
- No external config files

## Style

- Pythonic, concise code
- No emojis
- Conventional commit messages
